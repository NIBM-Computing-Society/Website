About AWS Cloud Club

AWS Cloud Clubs are student-led user groups that provide learning opportunities about cloud computing and career pathways. As an official AWS program, we bring together students passionate about cloud technologies.

  • Hands-on AWS workshops and labs
  • AWS certification preparation
  • Real-world cloud projects
  • Industry expert sessions

AWS Learning Paths

Structured learning paths to guide you from cloud beginner to AWS certified professional

1

Cloud Practitioner

Start your cloud journey with AWS fundamentals, core services, and basic architecture concepts.

  • • AWS Core Services (EC2, S3, IAM)
  • • Cloud Concepts & Pricing
  • • Security & Compliance Basics
  • • AWS Cloud Practitioner Certification
2

Solutions Architect

Design distributed systems on AWS. Learn about high availability, scalability, and cost optimization.

  • • AWS Architecture Best Practices
  • • Networking & Security
  • • Database Services & Analytics
  • • Solutions Architect Associate
3

DevOps & Advanced

Master DevOps practices, automation, and advanced AWS services for professional development.

  • • CI/CD with AWS Code*
  • • Containerization & Serverless
  • • Monitoring & Optimization
  • • DevOps Professional
